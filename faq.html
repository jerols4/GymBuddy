Gym Buddy was made using HTML, CSS, Javascript, and some custom Bootstrap! It is made by Kevin DeMaioribus, Jeremy Olin, Shane Keating, and Tyler Olin. Look at the code.

Currently, Gym Buddy is in a test mode with custom data from its creators. This is because we do not currently have any access to actual gym ID scanner data, so we have to make up our own for the time being.

Gym Buddy was made for SteelHacks 2020, a hackathon in Pittsburgh, Pennsylvania.
